# Map (Interactive Map Application)

## Description
Интерактивное мобильное приложение на Android, которое превращает статичное изображение (`map.jpg`) в полноценную навигационную карту. Приложение поддерживает плавное масштабирование (pinch-to-zoom) и перемещение по изображению для детального изучения. Пользователи могут взаимодействовать с маркерами для просмотра описаний объектов, а также добавлять собственные точки интереса долгим нажатием на экран. Все созданные метки сохраняются в локальном хранилище и остаются доступными после перезапуска приложения.

## Installation
Для локальной установки и запуска проекта выполните следующие шаги:
1. Клонируйте репозиторий с помощью команды:
   `git clone https://github.com/ваш-репозиторий/pmvs10b-project-coolmap.git`
2. Откройте проект в **Android Studio** (рекомендуется версия Ladybug или новее).
3. Убедитесь, что в папке `app/src/main/res/drawable/` присутствует файл `map.jpg`.
4. Дождитесь завершения синхронизации **Gradle**.
5. Запустите приложение на физическом устройстве или эмуляторе (API 24+).

## Usage
После запуска приложения вы увидите основное изображение карты. 
* **Навигация**: Используйте жест «щипок» двумя пальцами для приближения/удаления и перетаскивание одним пальцем для перемещения по карте.
* **Просмотр информации**: Нажмите на любой желтый маркер, чтобы открыть диалоговое окно с названием и подробным описанием локации.
* **Добавление точек**: Удерживайте палец на любой области карты (Long Press), чтобы вызвать меню создания новой точки. Введите название и описание, после чего точка будет сохранена.
