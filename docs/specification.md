# Дополнительная спецификация

## Ограничения
- Минимальная версия Android: API 24 (Nougat).
- Требуется поддержка мультитач для жестов масштабирования.
- Изображение карты должно быть в формате JPG или PNG и находиться в ресурсах приложения.

## Надежность
- Приложение обрабатывает ошибки парсинга JSON при загрузке сохраненных точек. Если данные повреждены, загружается список по умолчанию.
- Состояние масштаба и смещения сбрасывается при перезапуске (по умолчанию 1.0x).

## Безопасность
- Приложение не требует разрешений на доступ к интернету или личным данным пользователя.
- Данные о точках сохраняются в приватном хранилище SharedPreferences, доступном только данному приложению.
