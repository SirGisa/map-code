# Функциональные требования

## Описание требований
1. **Отображение карты**: Приложение должно загружать изображение `map.jpg` и отображать его как подложку карты.
2. **Масштабирование**: Пользователь может приближать и отдалять карту жестом "pinch-to-zoom".
3. **Панорамирование**: При приближении пользователь может перемещаться по карте с помощью свайпов.
4. **Информационные точки**: На карте отображаются предустановленные маркеры. При нажатии на маркер выводится диалоговое окно с заголовком и описанием.
5. **Создание меток**: Долгое нажатие на любую область карты открывает форму создания новой метки.
6. **Сохранение данных**: Все созданные пользователем метки должны сохраняться в памяти устройства и восстанавливаться при следующем запуске.

## Текстовые сценарии
### Сценарий 1: Просмотр информации о точке
1. Пользователь находит на карте маркер (желтый круг).
2. Пользователь нажимает на маркер.
3. Система открывает диалоговое окно с описанием данной локации.
4. Пользователь нажимает "Закрыть", окно исчезает.

### Сценарий 2: Добавление новой точки
1. Пользователь выбирает место на карте и зажимает палец на 1-2 секунды.
2. Появляется диалоговое окно "Добавить новую точку".
3. Пользователь вводит название и описание.
4. Пользователь нажимает "Добавить".
5. На карте в месте нажатия появляется новый маркер. Данные сохраняются в SharedPreferences.
