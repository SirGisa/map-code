# Документация проекта "Интерактивная карта"

## Краткое описание задачи
Цель проекта — разработка мобильного приложения для ОС Android, представляющего собой интерактивную карту на базе статического изображения. Приложение позволяет пользователям ориентироваться в пространстве вымышленного или реального мира, представленного на картинке, изучать точки интереса и добавлять свои собственные.
